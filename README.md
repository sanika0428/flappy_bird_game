# Flappy Bird Game
## Overview
This project is a simple implementation of the classic Flappy Bird game. The player controls a bird that must navigate through gaps between pipes without colliding with them. The game keeps track of the player's score based on how many obstacles are successfully passed.

## Features
Interactive Flappy Bird gameplay<br>
Score tracking system<br>
Collision detection with pipes and ground<br>
Simple and responsive controls<br>

## Technologies Used
Python<br>
Pygame<br>

## How to Run
Install Python on your system.<br>
Install the required library: pip install pygame<br>
Run the main game file: python flappy_bird.py<br>

## Controls
Press Spacebar or up arrow to make the bird jump.<br>
Avoid hitting pipes or the ground.<br>
